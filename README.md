## Overview

**HydroGrow** is an intelligent hydroponic fodder production system designed specifically for conditions in **Sri Lanka**. It integrates **Internet of Things (IoT)** sensors, **Deep Learning**, and **Machine Learning** techniques to optimize fodder growth, monitor environmental parameters in real-time, and improve yield and resource efficiency in soilless agriculture.

The system aims to address challenges in traditional fodder production such as land scarcity, climate variability, water usage, and labor intensity by enabling automated, data-driven control of hydroponic environments.

## Project Components

- **IoT Sensor Integration**: Real-time monitoring of critical parameters (pH, temperature, humidity, TDS/EC, light intensity, etc.).
- **Machine Learning & Deep Learning**: Predictive models for nutrient management, plant health monitoring, growth stage prediction, and anomaly detection.
- **Automation**: Actuators for automated nutrient dosing, water circulation, lighting, and climate control.
- **Web Interface**: A simple frontend (`index.html`) for visualization and monitoring (can be expanded into a full dashboard).
- **Documentation**: Detailed project report included as PDF.

## Features

- Real-time environmental monitoring via IoT
- Predictive analytics using ML/DL models
- Optimized nutrient delivery for faster fodder growth
- Reduced water and resource consumption
- Suitable for controlled environment fodder production (e.g., for livestock in Sri Lanka)
- Scalable for small to medium-scale farms

## Technologies Used

- **Hardware**: IoT sensors (pH, EC/TDS, temperature, humidity, light), microcontrollers (Arduino/ESP32/Raspberry Pi likely), actuators
- **Software**: Python (for ML/DL), TensorFlow/Keras or PyTorch (deep learning), Scikit-learn, IoT platforms (MQTT, Node-RED, etc.)
- **Frontend**: HTML/CSS/JS (expandable to React or Flask/Django dashboard)
- **Data Handling**: Real-time data logging and analysis
